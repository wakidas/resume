- **プロジェクト規模：**

- **使用技術：**
    - HTML、CSS(SCSS)、jQuery

**担当プロジェクト：**
1. **不動産ポータルサイト『SUUMO』賃貸資料請求画面エンハンス**
    - ABテストのフロント実装
